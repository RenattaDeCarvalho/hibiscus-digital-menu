## 🎯 Project Overview

Hibiscus Digital Menu is a modern digital menu solution designed to improve the customer experience through QR Code access, intuitive navigation and multilingual support.

The platform allows customers to browse products, explore categories and subcategories, search for items and view detailed product information in a fast and responsive interface.

---

## ✨ Features

- 🌍 Multilingual support (Portuguese, English and Spanish)
- 📱 Mobile-first responsive design
- 🔍 Product search
- 📂 Category and subcategory navigation
- 🍷 Wine catalog organization by country and type
- 🖼️ Product images and descriptions
- ⚡ Fast and optimized user experience
- 📱 QR Code ready
- 🎨 Modern and intuitive interface
